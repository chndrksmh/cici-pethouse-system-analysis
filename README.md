# cici-pethouse-system-analysis
Business analysis, UML design, testing documentation for pet grooming scheduling system.

Project sistem penjadwalan grooming berbasis web untuk digitalisasi proses booking.

Peran Saya:
- Business Analyst
- QA Tester
- System Analyst

Kontribusi:
- Observasi proses bisnis manual
- Requirement gathering
- Pembuatan UML
- Penyusunan test case
- Functional testing
- Bug reporting
- Validasi kebutuhan user

Catatan:
Source code tidak disertakan karena dikembangkan oleh tim/developer lain.
